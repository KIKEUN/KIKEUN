무엇이 달라졌나요

GPT-5.5는 단순히 더 똑똑한 것을 넘어, 더 적은 토큰과 재시도로도 높은 품질의 결과를 내는 효율적인 모델입니다.

#3

20시간도 끝까지 갑니다

복잡한 코딩 작업에서 멈추지 않고 끝까지 갑니다.
터미널 벤치 82.7%로 역대 최고 점수를 기록했고,

Terminal Bench(터미널 벤치)란?
AI가 실제 개발자처럼 터미널(명령어 창)을 얼마나 잘 다루는지를 평가하는 벤치마크

#4

문서부터 데이터까지 ‘실무형 AI’

Codex에서 GPT-5.5는 문서·스프레드시트·프레젠테이션 생성 능력이 향상되었으며, OpenAI 내부에서도 85% 이상이 매주 활용하고 있습니다. 재무팀은 리포트 자동화로 주당 5~10시간을 절약하고 있습니다.

Codex란?
코드 작성, 수정, 실행, 테스트까지 직접 처리하는 AI 에이전트이고
여러 파일·도구를 넘나들면서 문서, 엑셀, 자동화 작업까지 실제 업무를 진행합니다.

#5

연구도 몇 시간으로

Jackson Laboratory의 Derya Unutmaz 교수는 GPT-5.5 Pro를 활용해 62개 샘플과 약 2.8만 개 유전자로 구성된 유전자 발현 데이터를 분석하고, 연구 보고서와 핵심 인사이트까지 도출했으며, 이는 기존에는 연구팀이 수개월 걸릴 작업이라고 평가했습니다.
이러한 예시는 GPT‑5.5가 전문가의 의도를 실제로 작동하는 리서치 도구와 분석으로 구현하는 모습을 보여줍니다.

#6

절반 비용, 같은 결과

같은 문제를 풀 때 토큰을 훨씬 적게 쓰면서 더 좋은 결과를 냅니다.
코딩 인덱스 기준 경쟁 모델의 절반 비용으로
최상위 성능을 보여주고 있어요.

인공지능 분석 인텔리전스 지표(새 창에서 열기)는 외부 기관이 수행한 10가지 평가인 AA-LCR, AA-Omniscience, CritPt, GDPval-AA, GPQA Diamond, Humanity’s Last Exam, IFBench, SciCode, Terminal-Bench Hard, τ²-Bench Telecom의 가중 평균입니다.

#7

관련 종사자들은 이렇게 말해요

테스트 엔지니어들은 GPT-5.5가 GPT-5.4와 Claude Opus 4.7보다 추론 능력과 자율성이 뛰어나, 별도 지시 없이도 문제를 미리 파악하고 거의 완성된 결과를 만들어낸다고 평가했습니다.
엔지니어들은 GPT-5.5 없이는 업무 효율이 크게 떨어질 정도라고 표현하기도 했습니다.
